格式建議：MP3 / M4A / WAV / OGG
大小建議：單檔 20MB 以下
命名建議：好友名字.mp3（例如 grandma.mp3）

設定方法：
  在 src/data/wishes.js 中，將對應好友的
  media: null
  改為
  media: './audio/wishes/你的檔名.mp3'

注意：audio 會顯示自訂播放器（非影片畫面）。
      完全離線可用，不需網路。
