【Wishes Gacha 好友頭像放這裡】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

格式建議：JPG / PNG（正方形最佳）
尺寸建議：200×200px 以上（會裁切成圓形顯示）
命名建議：好友名字.jpg（例如 xiaoming.jpg）

設定方法：
  在 src/data/wishes.js 中，將對應好友的
  avatar: null
  改為
  avatar: './images/wishes/你的檔名.jpg'
