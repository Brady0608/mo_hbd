【Adventure Map 回憶照片放這裡】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

格式建議：JPG / PNG / WebP
尺寸建議：寬 600px × 高 400px（或 4:3 / 16:9 皆可）
命名建議：年份-事件.jpg（例如 2019-meet.jpg）

設定方法：
  在 src/data/memories.js 中，將對應節點的
  photo: null
  改為
  photo: './images/memories/你的檔名.jpg'
